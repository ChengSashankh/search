Deeno is a movie search engine 🔎 powered by Wikidata 📚, using advanced language models 🤖, vector databases and low-latency autocomplete ⚡️ to serve search.

The eventual goal is to build this for ALL of Wikipedia. Right now it just serves a small set of movies.

Built with:
Angular 15 Java Python HuggingFace Scala
